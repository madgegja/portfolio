# Chongsun Yu — Portfolio

Personal portfolio site for **Chongsun Yu**, Global Platform Operations Team Lead at Toomics Global.

## Live Site

**[madgegja.github.io/portfolio](https://madgegja.github.io/portfolio/)**

## Features

- **EN/KR Toggle** — Full bilingual support with localStorage persistence
- **4 Project Cards** — AI Agent, CS Chatbot, CS Analysis, BizOps Portal
- **Resume Download** — 2026 EN/KR DOCX
- **Scroll Animations** — IntersectionObserver-based reveal, counter, skill bars
- **Responsive** — Mobile-first, works on all screen sizes
- **Light Theme** — Clean, professional design

## Tech Stack

- Pure HTML / CSS / JavaScript (no build tools)
- Google Fonts (Inter + Noto Sans KR)
- GitHub Pages

## Structure

```
portfolio/
├── index.html              # Single-page portfolio
├── css/style.css           # Design system & responsive styles
├── js/main.js              # i18n toggle, animations, navigation
├── assets/docs/            # Resume DOCX files
└── README.md
```
